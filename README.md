# Game Vault – Assignment 4

## Overview
Game Vault is a React application created for COSC Assignment 4. This project demonstrates a shared team repository, React Router, and an API connection using the RAWG Video Games Database API.

## Features
- React + Vite
- React Router navigation
- Home, Dashboard, and Team pages
- API data fetched from RAWG
- Team collaboration using GitHub
